## Getting Started

    # Mongodb
    - You have to create a database to save our data.

    # Install Dependencies
    - $ npm install express
    - $ npm install prettier
    - $ npm install mongoose
    - $ npm install nodemon

    # Starting Server
    - $ nodemon start:dev
    - You will see the server on localhost link, there you have to use Postman or Insomnia.
    - http://localhost:8000
